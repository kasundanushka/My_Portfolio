A Python-based security tool designed to scan and identify vulnerabilities in local home networks, leveraging ethical hacking principles to enhance network protection.

This personal project was developed to enhance home network security using knowledge from my Introduction to Networking and Ethical Hacking diplomas. HARDWARE PROJECT

Automation Museum Audio Guide System

An automated audio guide system using microcontroller technology and sensors to provide location-based audio information, creating interactive museum experiences.

This hardware project combines embedded systems programming with IoT concepts to revolutionize museum visitor experiences. The system uses proximity sensors to detect visitor locations and automatically plays relevant audio descriptions for nearby exhibits. Key features include multilingual support, low-power operation for extended battery life, and easy content management for museum staff. The project demonstrates practical integration of hardware components, sensor technology, and audio processing for real-world applications.

TECHNOLOGIES

Arduino/Microcontroller, C/C++, Proximity Sensors, Audio Modules, IoT
