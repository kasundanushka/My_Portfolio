# My_Portfolio

Welcome to **My_Portfolio**!  
This is a personal portfolio website to showcase your projects, skills, experience, and contact information.

## Features

- Responsive design for mobile and desktop
- Sections for About, Projects, Skills, and Contact
- Easy navigation and customization
- Built with modern web technologies (edit this section if you used specific frameworks like React, Vue, or plain HTML/CSS/JS)

## What it does

**My_Portfolio** allows you to display your professional profile, highlight your work, and make it easy for visitors to get in touch with you. Customize it with your own details, images, and links to make it unique!

## Installation

### 1. Download the Project

Click the green `Code` button on this repository and select **Download ZIP**.  
Or clone the repository using:

```bash
git clone https://github.com/kasundanushka/My_Portfolio.git
```

### 2. Navigate to the Project Folder

```bash
cd My_Portfolio
```

## Running the Portfolio

Depending on how your project is made, follow the appropriate instructions:

### If built with HTML/CSS/JavaScript only:

1. Locate the main `index.html` file in the project folder.
2. Double-click on `index.html` to open it in your browser.

### If built with Node.js, React, or another framework:

1. Make sure you have [Node.js](https://nodejs.org/) installed.
2. Install required packages (if applicable):

    ```bash
    npm install
    ```

3. Start the local server:

    ```bash
    npm start
    ```

4. Open your browser and go to `http://localhost:3000` (or the port shown in your terminal).
